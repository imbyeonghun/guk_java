쇼핑몰 사이트 요구사항 명세서

의류 쇼핑몰

분류 관리
	분류 등록
		관리자는 등록된 제품의 분류를 등록할 수 있다
		분류 등록 시 분류명을 입력하며, 중복된 분류명을 사용할 수 없다
	분류 수정
		관리자는 등록된 제품의 분류를 선택하여 수정할 수 있다
		분류 수정 시 이미 존재하는 분류명을 사용할 수 없다
	분류 삭제
		관리자는 등록된 제품의 분류를 선택하여 삭제할 수 있다

제품 등록
	관리자는 제품 정보를 입력해서 제품을 등록할 수 있다
	제품 정보에는 제품 코드, 제품명, 분류, 제품 상세내용, 제품 가격, 썸네일으로 구성된다
	분류는 대분류만 적용하고 대분류는 옷, 신발, 모자, 악세서리, 기타로 구분
	제품 썸네일은 이미지이며, 최대 5개 선택이 가능하고, 최소 1개를 선택해야한다

제품 수정
	관리자는 등록된 제품 정보를 수정할 수 있다
	수정 가능한 제품 정보는 모든 제품 정보이다

제품 삭제
	관리자는 등록된 제품 정보를 삭제할 수 있다

제품 조회
	사용자는 분류를 이용해서 원하는 분류에 맞는 제품들을 조회할 수 있다
	사용자는 검색을 통해 검색어가 포함된 제품들을 조회할 수 있다

제품 상세 조회
	사용자는 제품 조회에서 조회한 제품들을 선택해서 상세 조회할 수 있다
	상세 조회 시 제품 썸네일들이 보여지며, 제품 정보들을 조회할 수 있다

장바구니
	제품 추가
		회원은 제품을 상세 조회해서 원하는 제품을 장바구니에 담을 수 있다
			제품에 수량을 선택해서 장바구니에 담을 수 있다
			이미 추가된 제품을 장바구니에 담으면 수량이 변경
	제품 삭제
		회원은 장바구니에 담긴 제품을 삭제할 수 있다
	제품 수정
		회원은 장바구니에 담긴 제품의 수량을 수정할 수 있다

제품 구매
	바로 구매
		회원은 제품 상세 조회에서 제품의 수량을 선택해서 바로 구매를 통해 한 제품을 구매할 수 있다
	장바구니 구매
		회원은 장바구니에 담은 제품들 중에서 선택해서 여러 제품을 한번에 구매할 수 있다
		여러 제품을 한번에 구매할 때 각 제품마다 구매 번호가 할당이 된다

구매 취소
	회원은 구매 제품 내역에서 배송 전인 제품에 대해 구매를 취소할 수 있다
	취소 시 결제한 금액을 환불한다
	취소 시 구매 취고한 제품을 구매 목록에서 제거한다
	취소 시 구매 취소한 제품은 구매 취소 목록에 추가한다

제품 환불
	회원은 배송 된 제품에 문제가 있을 시 제품을 환불받을 수 있다
	제품 환불 시 환불한 제품이 판매자에게 배송이 되고, 
		제품 불량이 확인이 되면 관리자는 해당 제품을 환불한다
	환불한 제품은 구매 목록에서 제거되고, 환불 목록에 추가된다

제품 교환
	회원은 배송 된 제품에 문제가 있을 시 제품을 교환 받을 수 있다
	제품 교환 시 교환한 제품이 판매자에게 배송되고, 제품 불량이 확인되면 관리자는 제품을 교환해준다

구매 확정
	회원은 배송된 제품에 문제가 없는 경우 구매 확정을 할 수 있다
	구매 확정 시 환불,교환이 불가능하다

회원 관리
	회원 가입
		사용자는 아이디, 비번, 이메일, 전화번호를 입력하여 회원가입을 할 수 있다
		아이디는 영문자, 숫자만 가능하며 8~13자이다
		아이디는 중복 검사를 통해 중복되지 않은 경우에만 사용할 수 있다
		비밀번호는 영문자, 숫자, 특수문자(!@#$%)만 가능하며, 8~20자이다
		이메일은 이메일 형식에 맞게 작성한다
		전화번호는 전화번호 형식에 맞게 작성한다
		아이디, 비번, 이메일, 전화번호는 필수 항목이다
	
	로그인
		사용자는 아이디, 비번을 입력하여 로그인할 수 있다
		아이디, 비밀번호가 일치하면 로그인이 된다
		아이디, 비밀번호가 일치하지 않으면 알림 메시지를 띄운다
		로그인 5회 실패 시 비밀번호 찾기를 통해 비밀번호를 재발급 받아야 로그인이 가능하다
		로그인 성공 시 실패 횟수가 0으로 초기화된다
	
	로그아웃
		회원은 로그아웃할 수 있다
	
	비밀번호 찾기
		회원은 아이디와 이메일을 입력하여 비밀번호를 찾을 수 있다
		입력한 아이디와 이메일이 회원가입 시 입력한 아이디, 이메일과 일치하면 메일로 비밀번호를 변경할 수 있는 페이지의 링크를 전송한다
			전송된 링크는 유효시간이 5분이다
			?id=아이디&code=인증코드
		일치하지 않으면 알림 메시지를 띄운다
		이메일에서 보낸 링크를 클릭하면 새 비밀번호를 변경할 수 있는 페이지로 이동한다
		이동한 페이지에서 새 비밀번호와 새 비밀번호 확인을 입력하면 비밀번호가 변경된다

	마이페이지
		회원은 마이페이지에서 회원 정보를 확인할 수 있다
		마이페이지에서는 구매 목록, 구매 취소 목록, 반품 목록, 환불 목록, 구매 완료 목록을 조회할 수 있다
		회원 정보를 수정할 수 있다
			회원 정보는 비밀번호, 이메일, 전화번호를 수정할 수 있다

릴레이션 스키마

회원(아이디(Pk), 비밀번호, 이메일, 전화번호, 권한, 실패횟수)
인증정보(인증번호(PK), 인증코드, 만료기간, 아이디(FK))
제품(제품코드(PK), 제품명, 내용, 가격, 분류번호(FK))
분류(분류번호(PK), 분류명)
주문(주문번호(PK), 주문일, 수량, 주문 상태, 결제액, 아이디(FK), 제품코드(FK))
장바구니(장바구니번호(PK(대리키)), 수량, 아이디(FK), 제품코드(FK))
썸네일(썸네일번호(PK(대리키)), 파일명, 제품코드(FK))
